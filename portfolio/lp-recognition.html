License Plate Recognition using Computer Vision and OCR

This project focuses on the development of an Automatic License Plate Recognition (ALPR) system that utilizes pre-trained YOLOv8 models and the EasyOCR library to extract alphanumeric data.

Project Overview

The goal was to build an ALPR system capable of recognizing license plates from various countries under different lighting conditions.

Challenges and Solutions

Challenges included accurately detecting plates in low light and differentiating between varying formats. The solution involved using pre-trained YOLOv8 models to improve plate detection.

Technologies and Tools Used

Technologies: Computer Vision, Optical Character Recognition, YOLO

Tools: Python, OpenCV, EasyOCR, YOLOv8
