The Multifamily Real Estate Development Process

This project presents an optimization framework to evaluate the feasibility of multifamily real estate development. Using data from Austin, TX, the framework helps predict the optimal number of units that are likely to receive approval from city councils based on characteristics of nearby properties.

Project Overview

The goal of the project was to create an optimization tool that evaluates the development potential of land parcels in Austin, TX, and predicts the number of residential units likely to receive approval.

Challenges and Solutions

The biggest challenge was identifying the most critical features influencing the development approval process. We addressed this by analyzing previously approved developments and using machine learning to determine feature importance.

Technologies and Tools Used

Technologies: Optimization, Real Estate Development, Data Analysis

Tools: Python, Scikit-Learn, Linear Programming

Results

The optimization model provided accurate predictions for the number of units likely to be approved, offering valuable insights for multifamily real estate developers during the feasibility analysis stage.
