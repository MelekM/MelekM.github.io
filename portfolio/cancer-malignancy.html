Breast Cancer Malignancy Diagnostics using Neural Networks

This project applies multiple statistical learning techniques to model breast cancer malignancy using cellular nuclei measurements. It compares classical statistical methods and deep neural networks to improve diagnosis accuracy, with careful parameter tuning yielding the highest accuracy using neural networks.

Project Overview

The primary goal was to model and predict breast cancer malignancy based on medical data from cellular nuclei. The project involved comparing classical statistical learning techniques with modern neural networks to identify the most effective model for diagnostic purposes.

Challenges and Solutions

Challenges included finding the optimal model architecture and tuning hyperparameters for the neural networks. We addressed this by using grid search for hyperparameter optimization and testing multiple neural network architectures.

Technologies and Tools Used

Technologies: Data Analysis, Neural Networks, Medical Diagnostics

Tools: Python, TensorFlow, Scikit-Learn, Pandas

Results

The deep neural network model outperformed classical methods, yielding the highest diagnostic accuracy after careful tuning of the parameters and model structure.
