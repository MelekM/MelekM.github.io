Washington State Algorithmic Redistricting

In this project, I collaborated on the development of an optimized redistricting plan for Washington State. Using algorithmic methods, we aimed to balance population distribution while maintaining electoral fairness.

Project Overview

The goal was to apply machine learning techniques to optimize electoral redistricting and ensure fairness across voting districts in Washington State.

Challenges and Solutions

The challenge was to create districts that balance population while avoiding political bias. We used unsupervised learning techniques to remove human bias and ensure the best results.

Technologies and Tools Used

Technologies: Machine Learning, Algorithmic Optimization

Tools: Python, SciPy, Pandas

Results

We developed an optimized redistricting plan that reduced population disparities between districts while adhering to legal boundaries.
